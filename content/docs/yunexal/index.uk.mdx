# Yunexal Panel

**v0.5.0** — self-hosted платформа керування серверами, побудована на Docker.

Yunexal Panel використовує **Rust + Axum**, **SQLite** і **Bollard**. Templates і static assets компілюються в один binary, тому для runtime не потрібні окремі frontend-файли.

## Статус репозиторію

Робота над installer-image customization зараз призупинена. Repository scripts лишаються local-only файлами та ігноруються git, тому не входять у GitHub-tracked source set.

Активний і підтримуваний сценарій — direct host installation через release binaries і `yunexal-setup`.

## Що вже є

- Dashboard із live container cards, CPU/RAM/Network/Disk metrics, uptime, status badges і `My only` filter
- WebSocket console через `xterm.js`, ANSI colors, command input, live charts і storage card
- SPA-style server tabs: Console, Files, Networking, Settings, Audit, Users
- File manager із breadcrumbs, 150+ icons, Ace editor, uploads, archive/extract і safe path handling
- Server settings для ENV, factory reset і admin-only delete
- Server members із UID lookup і policies `none / read / write`
- Immutable audit logs із filters, Device column, pagination і download
- Networking: port mappings, labels, enable/disable, UFW blocks і Linux `tc` bandwidth limits
- Container creation із local-first images, Dockerfile build, Compose-style YAML, ENV fetch, owner assignment і quota preflight
- Admin Panel: Overview, Containers, Images, Users, Roles, Audit Log, Panel Settings
- Session devices, password-change invalidation, service API login, encrypted cookies, Argon2id, rate limits і security headers
