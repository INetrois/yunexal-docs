# Admin і Updates

Admin Panel містить platform controls для single-host Docker management.

## Admin tabs

Документовані v0.5.0 admin tabs:

- Overview
- Containers
- Images
- Users
- Roles
- Audit Log
- Panel Settings

## Core admin capabilities

- Users: create, delete, set passwords, UID + Nickname identities і tri-state RBAC roles
- Roles & Permissions: custom roles, `read / none / write` policies, role colors, immutable `root`
- Images: pull, delete, duplicate і ENV override editor
- Containers: disk limit, bandwidth, ENV, ports, stop all, row state updates without animation flicker
- Audit Log: immutable records, 200 per page, multi-select action filter, parsed Device column і full User-Agent tooltip
- Panel Updates: stable/unstable checks, download/install і auto-restart
- Panel Settings: root-only categories Storage, Security, Operations, Interface

## Upcoming admin areas

Dispatchers, firewall-wide rules, backups, tickets, insights, notifications, themes, marketplace, plugins, localization і accessibility є roadmap items, а не completed admin surfaces.
