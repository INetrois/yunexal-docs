# Features

Yunexal Panel `v0.5.0` covers day-to-day Docker server management: live dashboards, console access, file operations, networking, server-scoped access control, audit logs, container creation, and root-level administration.

<Cards>
  <Card title="Dashboard And Console" href="/docs/yunexal/features/dashboard-console">
    Live cards, metrics, device settings, WebSocket TTY attach, charts, and storage information.
  </Card>
  <Card title="File Manager" href="/docs/yunexal/features/file-manager">
    Breadcrumbs, 150+ file icons, Ace editing, streamed uploads, archives, and safe path enforcement.
  </Card>
  <Card title="Networking" href="/docs/yunexal/features/networking-dns">
    Port mappings, labels, enable/disable controls, UFW blocks, and Linux `tc` bandwidth limits.
  </Card>
  <Card title="Admin And Updates" href="/docs/yunexal/features/admin-and-updates">
    Overview, containers, images, users, RBAC roles, audit log, panel settings, and one-click updates.
  </Card>
